# mediaCentral
mediaCentral adalah media yang menggabungkan berbagai berita indonesia dari berbagai sumber media berita.

###How to use
- Clone master branch
- npm install
- change apikey in services/configs/config.js
- npm start

###What inside this project
- axios
- react-icons
- react-router
- bootstrap
- sweet alert
